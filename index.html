Introduction to Programming Nanodegree STAGE 0
KEY TAKEAWAYS

THE BASICS
World Wide Web (WWW): A collection of HTML documents
HTML (HyperText Markup Language): The basis for almost every web page. It's what glues everything together.

COMPONENTS OF THE WEB
HTTP: The main protocol of the web used to communicate with you, your browser, and servers
Browser: Program that runs on your computer to display files on the web
Internet: World's largest computer network
Servers: Computers that host the files that make up the web

HTML BASICS
Made up of a series of tags
Tags: Are always contained within angled brackets. Most have an opening and closing tag.
Void tags: Do not have end tags. (examples are <img> and <br>)
Elements: refers to everything within a set of opening and closing tags
Inline vs Block elements Block elements form an "invisible box" around the content inside of them. Inline do not.
Attributes: a value within an HTML tag. An example is the <a> anchor tag

STRUCTURE OF AN HTML DOCUMENT
<!doctype html>
<html>
<head>(meta, css, JS)
<title> Title </title>
</head>
<body>
<b>content</b>
</body>
</html>
